Collection of tools to configure Dell Remote Access Controllers (DRAC)

## Installation

Install from PSGallery

```powershell
PS> Install-Module -Name DellRacTools
```

## Getting Started

Get a list of available commands

```powershell
PS> Get-Command -Module DellRacTools
```
